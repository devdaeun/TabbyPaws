# Tabby Paws

고양이 관련 정보전달 및 쇼핑몰 웹사이트 프로젝트입니다.

## 프로젝트 소개
- 고양이 전용 사료 쇼핑몰 및 정보전달 웹 사이트 프로젝트.
- 사료에 대한 성분정보를 페이지 내부에서 번거롭게 찾아봐야하는 경우가 많아 주 성분을 바로 확인할 수 있는 쇼핑몰을 구현하고자 함.

1. 신상품 자동 업데이트 및 추천
2. 상품 카테고리 검색기능
3. 사료 테스트 및 추천기능
